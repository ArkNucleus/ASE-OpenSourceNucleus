# Ark-Nucleus
 Ark: Survival Evolved Game Mod with the functionality to manage other compatible mods and help admins in managing servers.

 - Clone the files into a "Ark-Nucleus" named folder inside your ASE DevKits Mods Folder while the DevKit is Closed.
 - Open Devkit and load in the Mod
 - Rename or Copy the folder under a different name before starting to work on your own modified version.
 - Replace all instances of "Ark-Nucleus" test or Art in the mod with your own Mod Name.
